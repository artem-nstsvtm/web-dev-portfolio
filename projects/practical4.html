What is selector’s specificity and how it’s calculated?

The selector's specificity is a value that determines which style rule will be applied when multiple conflicting rules target the same element. Specificity is calculated based on the combination of selectors used in the CSS rule. The more specific a selector is, the higher its specificity value.

What are complex selectors?

Complex selectors are CSS selectors that involve combining multiple simple selectors to target elements in a more specific way. Examples of complex selectors include descendant selectors, child selectors, adjacent sibling selectors, and more.

What is the difference between pseudo-classes and pseudo-elements?

Pseudo-classes are keywords that specify a special state of the selected element, such as ":hover" for mouse-hover effect.Pseudo-elements, on the other hand, target specific parts of an element, like "::before" and "::after" for inserting content before and after an element respectively.

Blockquotes

A block quotation (also known as a long quotation or extract) is a quotation in a written document, that is set off from the main text as a paragraph, or block of text.

It is typically distinguished visually using indentation and a different typeface or smaller size quotation. It may or may not include a citation, usually placed at the bottom.

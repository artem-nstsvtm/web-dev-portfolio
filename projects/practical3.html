Ways to Open Developer's Console

To open the developer's console in Mozilla Firefox, you can press F12 or right-click and select Inspect Element. In Safari, you can go to Develop, then Show Web Inspector.

Internal Links: Absolute vs. Relative Links

Relative links are more preferable to be used as they are more flexible when you need to move a website into another domain. Absolute links are the type of links where you use the full URL, while relative links lead to the files or pages from current location.

In other words use absolute links when you need to link to an external website, while relative links are used when you need to link within the same website.

Image Formats in Web Development

Image formats used in web development are JPG, GIF, PNG, WEBP, and SVG.

JPG (or JPEG) is great for photographs because of its compression abilities. GIF is used for animations and images with fewer colors. PNG is used for images with transparency.

WEBP is a modern image format developed by Google, offering the best compression and quality. SVG (Scalable Vector Graphics) is an XML-based vector image format that is ideal for logos and icons.

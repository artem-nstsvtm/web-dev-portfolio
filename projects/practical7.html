What is repaint and reflow of the webpage?

Repaint updates the visual appearance without changing layout, while reflow recalculates positions and sizes of elements.

How does JavaScript affect page load speed?

Poorly optimized JavaScript can slow down page loading time, impacting performance negatively, that's why developers often write JS code in additional file.

What is minification of scripts and its benefits?

Minification removes unnecessary characters from code, such as long variable names, white-spaces reducing file size and improving page loading speed.

What is a JavaScript Web Worker and when do we need it?

A JavaScript Web Worker is a script that runs in the background, separate from the main browser thread, it is useful for handling complex tasks without blocking the UI.
